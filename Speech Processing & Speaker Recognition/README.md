This repository contains an end-to-end speech processing lab project focused on **text-independent Speaker Recognition** using deep neural networks. The workflow covers everything from literature synthesis and audio data preprocessing to feature extraction, custom neural network design, evaluation, and empirical optimization.

## 🎙️ Personalized Speaker Profile & Voice Detection
Unlike general-purpose datasets, this repository features a highly personalized classification task. The model is specifically trained to recognize and validate **my personal voice** (`Mohammed Islem GOURINE`) against a control group of alternative speakers. 

During the validation and testing phases:
* **Target Speaker**: Mohammed Islem GOURINE (Verified voice profile).
* **Objective**: Achieve high-precision, text-independent verification to ensure the model correctly triggers a positive match for my voice while minimizing false acceptance rates (FAR) for other speaker profiles.
